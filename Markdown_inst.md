## Списки

Чтобы добавить ненумерованные списки необходимо пункты выделить (*). Например:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованный список необходимо просто пронумеровать. Например:
1. Первый пункт.
2. Второй пункт.
